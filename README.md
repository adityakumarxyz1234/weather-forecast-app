# 🌦️ Weather Forecast App

A modern and responsive Weather Forecast Web Application built using
**HTML, CSS, and JavaScript**. This application provides real-time
weather information based on the user's current location or a searched
city using a weather API.

## 🚀 Live Demo

**Netlify:** https://aditya-weather-forecast.netlify.app

## ✨ Features

-   🌍 Automatic weather detection using current location
-   🔍 Search weather by city name
-   🌡️ Real-time temperature display
-   💧 Humidity information
-   💨 Wind speed information
-   🌤️ Dynamic weather icons
-   🖼️ Weather-based background changes
-   📱 Fully responsive design
-   ⚡ Fast and lightweight

## 🛠️ Built With

-   HTML5
-   CSS3
-   JavaScript (ES6)
-   OpenWeather API

## ⚙️ Installation

1.  Clone the repository:

``` bash
git clone https://github.com/adityakumarxyz1234/weather-forecast-app.git
```

2.  Open the project folder:

``` bash
cd weather-forecast-app
```

3.  Open `index.html` in your browser.

## 🔑 API Setup

1.  Create a free API key from OpenWeather.
2.  Open `script.js`.
3.  Replace the API key:

``` javascript
const API_KEY = "YOUR_API_KEY";
```
